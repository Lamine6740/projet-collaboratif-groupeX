# 🍏 UNIVERS APPLE

## 📌 Description du projet
Ce projet est une application web de vente de produits Apple réalisée en équipe avec **Git et GitHub**.

L’objectif est de simuler un vrai environnement de travail où chaque membre possède un rôle précis.
