"# Mon projet" 
"## Organisation des branches" 


1-Description du projet
Ce projet est une application web de vente de produits Apple réalisée en équipe dans le cadre d’un travail collaboratif utilisant Git et GitHub.
L’objectif est de simuler un environnement professionnel avec plusieurs rôles travaillant ensemble sur un même projet.
